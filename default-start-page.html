Espace Client
Vous ne pouvez vous rendre dans votre Espace Client qu'après vous être identifé :  
http://client.amen.fr 
Vous aurez alors accès à toutes les informations concernant votre commande (statut de votre compte, recevoir vos mots de passe, facture ...).  

A travers cet Espace Client, vous pouvez également :
- Upgrader votre pack
- Augmenter l'espace disque
- Ajouter des comptes POP
- Créer un sous-domaine de votre nom de domaine
- Acheter un module de paiement sécurisé
- Modifier vos coordonnées personnelles
- Paramétrer vos méta-tags
- ...
